# Contributing to Growth4U

Thanks for your interest in contributing! Here's how you can help.

## Reporting issues

Found a bug or have a suggestion? [Open an issue](https://github.com/Growth4U-systems) in the relevant repository. Please include:

- A clear description of the problem or idea
- Steps to reproduce (for bugs)
- Your environment (OS, Node/Python version, etc.)

## Pull requests

1. Fork the repo and create your branch from `main`
2. Make your changes and test them
3. Write a clear PR description explaining what and why
4. Submit your pull request

## Code style

- Follow the existing patterns in the codebase
- Keep changes focused — one feature or fix per PR
- Write clear commit messages

## Questions?

Reach out at accounts@growth4u.io
